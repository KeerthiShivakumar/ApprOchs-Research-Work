# In-Memory & Approximate Computing Based Adder Design

## 📌Objective
This project focuses on analyzing and extending a research paper in the domain of **In-Memory Computing** and **Approximate Computing**, with an aim to propose improvements and develop a publishable research outcome.

## 📖 Work Done So Far

### Paper Study
- Read and thoroughly understood the research paper
- Presented the concepts to the guide
- Clarified core theoretical concepts

### Concept Exploration
- Studied **In-Memory Computing**
- Studied **Approximate Computing**

### Implementation
- Designed and implemented logic using **SystemVerilog**
- Simulated using **Xilinx Vivado**
- Tested for multiple values of parameter *k*

### Analysis

- Identified corner cases
- Compared:
  * Power reports
  * Utilization reports
  * Performance vs normal adders

### Hardware Implementation
- Successfully implemented design on **FPGA**

## 📊 Key Observations
- Trade-offs observed between accuracy and power
- Approximate approaches show improvement in resource usage
- Corner cases impact correctness under certain conditions
